# NodeJS REST API

> Full CRUD REST API using Node.js without any framework and libraries

```
# Routes

GET          /signup
POST         /signup
PATCH/PUT    /signup/:id
DELETE       /signup/:id

POST         /login

```
```

## Usage

```
# Install
npm install

npm start
